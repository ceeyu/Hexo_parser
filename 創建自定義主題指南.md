## 第二部分：iThome 文章導入方案

### 方案 1：手動導入（推薦，質量最好）

**優點：**
- 完全控制格式
- 可以優化內容
- 保留所有圖片和代碼

**步驟：**
1. 從 iThome 複製文章內容
2. 使用 Hexo 創建新文章
3. 貼上內容並調整格式
4. 添加 front-matter 元數據

**批量處理腳本：**
```javascript
// scripts/import-ithome.js
// 手動輔助工具，幫助快速創建文章
```

### 方案 2：使用爬蟲自動導入

**注意：** 需要遵守 iThome 的使用條款

**工具：**
- Puppeteer（瀏覽器自動化）
- Cheerio（HTML 解析）

### 方案 3：使用 RSS/API（如果有）

檢查 iThome 是否提供 RSS feed 或 API

## 第三部分：性能優化技巧

### 1. 圖片優化
- 使用 WebP 格式
- 延遲加載（Lazy Loading）
- 響應式圖片

### 2. CSS 優化
- Critical CSS 內聯
- 移除未使用的 CSS
- 使用 CSS 變量

### 3. JavaScript 優化
- 最小化使用
- 延遲加載非關鍵腳本
- 使用原生 API

### 4. 字體優化
- 使用系統字體棧
- 或使用 font-display: swap

### 5. 代碼高亮優化
- 使用輕量級方案（如 Prism.js）
- 或服務端渲染高亮
