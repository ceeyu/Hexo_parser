## 🚀 使用流程

### 日常更新

```cmd
# 1. 創建或爬取文章
node tools/快速創建文章.js
# 或
node tools/批量爬取系列文章.js

# 2. 本地預覽
npm run server

# 3. 部署並推送
tools\部署並推送.cmd "提交訊息"
```

### 快速推送

```cmd
tools\git-push.cmd "提交訊息"
```

### 格式修復

```cmd
node tools/修復文章格式.js
```
