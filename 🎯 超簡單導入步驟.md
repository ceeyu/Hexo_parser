# 🎯 超簡單的 iThome 文章導入步驟

忘掉複雜的工具，用最簡單的方式！

---

## 📝 完整流程（5 分鐘搞定一篇）

### 步驟 1：創建文章（30秒）

打開命令提示符，運行：
```cmd
node tools/快速創建文章.js
```

按提示輸入：
- **標題**：從 iThome 複製標題
- **日期**：直接按 Enter（使用今天）
- **標籤**：例如 `python, AI, 機器學習`
- **分類**：例如 `技術研究`

會顯示：
```
✅ 文章已創建！
📄 文件位置: C:\Users\...\source\_posts\2025-12-23-文章標題.md
```

### 步驟 2：打開 iThome 文章（10秒）

1. 前往：https://ithelp.ithome.com.tw/users/20151593/articles
2. 點擊你要導入的文章

### 步驟 3：複製內容（30秒）

在 iThome 文章頁面：
1. 滾動到文章內容開始的地方
2. 用滑鼠選取所有內容（從第一段到最後）
3. 按 `Ctrl + C` 複製

### 步驟 4：貼到 Hexo 文章（1分鐘）

1. 用記事本或任何編輯器打開步驟1創建的 `.md` 文件
2. 找到這部分：
   ```markdown
   ---
   title: 文章標題
   date: 2025-12-23
   tags: [...]
   categories: ...
   ---

   ## 簡介          ← 從這裡開始選取

   在這裡寫文章簡介...

   ## 內容

   ### 小標題 1

   內容...
   ```

3. **選取** `## 簡介` 及以下所有內容
4. **刪除**（Delete 鍵）
5. **貼上** iThome 的內容（Ctrl + V）
6. **保存**（Ctrl + S）

### 步驟 5：預覽（1分鐘）

```cmd
npm run server
```

打開瀏覽器訪問：http://localhost:4000

檢查文章是否正常顯示

### 步驟 6：部署到 AWS（2分鐘）

如果預覽沒問題：

```cmd
按 Ctrl+C 停止服務器
npm run clean
npm run build
npm run deploy
```

等待上傳完成，訪問你的網站！

## 💡 小技巧

### 快速編輯文章

**方法 1：用記事本**
```cmd
notepad source\_posts\文章檔名.md
```

**方法 2：用 VS Code（如果有安裝）**
```cmd
code source\_posts\文章檔名.md
```

**方法 3：直接在文件管理器**
1. 打開 `C:\Users\T8USER\Desktop\my-blog\source\_posts\`
2. 雙擊 `.md` 文件
3. 選擇用記事本打開

### 批量導入

一次導入多篇：
1. 創建 5 篇文章（運行 5 次創建命令）
2. 依次從 iThome 複製內容貼上
3. 全部完成後一次性部署

### 格式調整

如果貼上的內容格式有問題：

**代碼塊**：手動加上 \`\`\`
```markdown
\`\`\`python
你的代碼
\`\`\`
```

**標題**：手動加上 ##
```markdown
## 這是標題
```

**列表**：手動加上 -
```markdown
- 項目 1
- 項目 2
```

## 🆘 遇到問題？

### 找不到創建的文章文件？
位置在：`C:\Users\T8USER\Desktop\my-blog\source\_posts\`

### 預覽時文章沒顯示？
1. 檢查文件的 front-matter（--- 之間的部分）格式是否正確
2. 運行 `npm run clean` 清理緩存
3. 重新運行 `npm run server`

### 部署後網站沒更新？
1. 等待 1-2 分鐘
2. 清除瀏覽器緩存（Ctrl + F5）
3. 確認部署沒有錯誤訊息
