✅ Hexo 已安裝  
✅ 自定義主題已創建  
✅ AWS S3 已配置

## 📝 方法一：從 iThome 導入文章（推薦）

### 步驟 1：打開你的 iThome 文章

1. 前往：https://ithelp.ithome.com.tw/users/20151593/articles
2. 選擇一篇你想導入的文章
3. 打開文章頁面

### 步驟 2：創建新文章

在命令提示符（CMD）中運行：

```cmd
node tools/快速創建文章.js
```

按提示輸入：
- 📝 文章標題：（從 iThome 複製標題）
- 📅 發布日期：（直接按 Enter 使用今天，或輸入 YYYY-MM-DD）
- 🏷️ 標籤：（例如：python,AI,機器學習）
- 📁 分類：（例如：技術研究）

### 步驟 3：編輯文章內容

1. 腳本會告訴你文件位置，例如：
   ```
   C:\Users\T8USER\Desktop\my-blog\source\_posts\2025-12-23-文章標題.md
   ```

2. 用任何文本編輯器打開這個文件（記事本、VS Code、Notepad++ 都可以）

3. 從 iThome 複製文章內容

4. 貼到文件中 `## 簡介` 的位置，替換掉示例內容

5. 保存文件

### 步驟 4：預覽效果

如果服務器還在運行：
- 直接刷新瀏覽器 http://localhost:4000

如果服務器已停止：
```cmd
npm run server
```

### 步驟 5：部署到 AWS

確認效果沒問題後：

```cmd
npm run clean
npm run build
npm run deploy
```

等待上傳完成，訪問你的網站：
http://xian-hexo-blog-2025.s3-website-ap-northeast-1.amazonaws.com

## ✍️ 方法二：直接寫新文章

### 快速創建

```cmd
npx hexo new post "我的新文章標題"
```

### 編輯文章

1. 打開 `source/_posts/我的新文章標題.md`

2. 編輯 front-matter（文章開頭的 --- 之間的部分）：

```markdown
---
title: 我的新文章標題
date: 2025-12-23
tags: [標籤1, 標籤2]
categories: 分類名稱
---

## 這裡開始寫內容

你的文章內容...
```

3. 保存

### 預覽和部署

同方法一的步驟 4 和 5

## 🎨 自定義主題樣式

### 修改顏色

編輯 `themes/fast-theme/source/css/style.css`

找到這段：
```css
:root {
  --primary-color: #0066cc;  /* 改這裡改主色調 */
  --text-color: #333;        /* 文字顏色 */
  --bg-color: #fff;          /* 背景色 */
}
```

改成你喜歡的顏色，例如：
```css
:root {
  --primary-color: #ff6b6b;  /* 紅色 */
  --text-color: #2c3e50;     /* 深灰色 */
  --bg-color: #f8f9fa;       /* 淺灰背景 */
}
```

保存後刷新瀏覽器查看效果。

### 修改導航菜單

編輯 `themes/fast-theme/_config.yml`

```yaml
menu:
  首頁: /
  歸檔: /archives
  關於: /about
  # 添加更多菜單項
```

### 修改社交連結

同一個文件：
```yaml
social:
  GitHub: https://github.com/你的用戶名
  iThome: https://ithelp.ithome.com.tw/users/20151593
  # 添加更多連結
```

## ❓ 常見問題

### Q: 服務器怎麼停止？
A: 在命令提示符按 `Ctrl + C`

### Q: 修改了樣式但沒生效？
A: 
```cmd
npm run clean
npm run server
```

### Q: 文章沒有顯示？
A: 檢查文章的 front-matter 格式是否正確，特別是 `---` 符號

### Q: 部署後網站沒更新？
A: 
1. 確認部署成功（沒有錯誤訊息）
2. 清除瀏覽器緩存（Ctrl + F5）
3. 等待 1-2 分鐘（S3 需要時間同步）

### Q: 圖片怎麼添加？
A: 
1. 把圖片放到 `source/images/` 目錄
2. 在文章中使用：`![圖片說明](/images/圖片名稱.jpg)`
